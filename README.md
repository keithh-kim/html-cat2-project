# StrathCloud Solutions - Web Application Project

A fully responsive multi-page web application representing a premium DevSecOps and Cloud Migration consulting agency. Built using HTML5, CSS3, JavaScript (ES6+), and Bootstrap 5.

## Project Details
- **Author:** Keith Kimani Njunge
- **Start Date:** 15 June 2026
- **Target Pages:** 8 Interlinked Pages
- **Core Technologies:** Bootstrap 5, Custom CSS, Vanilla JavaScript

## Planned Interactive Features
1. Dynamic AWS/Cloud Engineering Cost Estimator
2. Live Filterable Case Studies Grid
3. Real-time Contact Form Input Validation
